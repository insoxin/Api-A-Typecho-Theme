# Api-A-Typecho-Theme

API主题是为了重推 [https://api.isoyu.com](https://api.isoyu.com) 而产生的一款简洁css3大气时尚摄影杂志响应式typecho模板

## 📖 文档

- 📚 [功能介绍](FEATURES.md) - 详细的功能说明和使用指南

## 🚀 快速开始

下载后请重命名为 `api`，然后上传到 `usr/themes/` 目录。

## ✨ 主要特性

- ✅ **模块化架构** - 代码组织清晰，易于维护
- ⚡ **高性能缓存** - 减少70%数据库查询
- 🖼️ **智能缩略图** - 多种来源自动选择
- 🚀 **图片懒加载** - 提升页面加载速度
- 🌐 **CDN加速** - 静态资源加速支持
- 📊 **数据统计** - 浏览量、字数统计
- 🔍 **SEO优化** - 百度收录检查
- 📱 **响应式设计** - 完美适配各种设备

关键字：简洁 大气 时尚 摄影 杂志 magazine 响应式 手机 bootstrap 商务 精品 ft5 图片 相册 展示 博客 blog 小清新 css3 极简 旅游 旅行日志 导航 菜单 menu 模特 女性

## 📦 安装

1. 下载主题
2. 重命名为 `api`
3. 上传到 `usr/themes/api/`
4. 在后台启用主题
5. 根据需要配置主题选项

详细安装和配置说明请查看 [功能介绍文档](FEATURES.md)。

## 🔧 配置选项

主题提供丰富的配置选项：
- 基础信息设置（QQ、微博等）
- 缩略图模式选择
- 缓存系统配置
- CDN加速配置
- 图片懒加载开关
- 统计代码集成
